# Demo 演示脚本

## 登录

- 打开 `https://your-domain.example`。
- 使用私下提供的 Demo 账号登录。

## 知识库

- 创建或打开 Demo 知识库。
- 上传一份安全示例文档，或使用预置文档。
- 等待解析、分块、向量化任务完成。

## 对话

先问：

> 用三点总结这份文档，并标出来源页码或来源片段。

再问：

> 对比文档中的两个章节，说明它们的区别。

## 评估

- 为 Demo 文档生成 QA 对。
- 运行评估任务。
- 展示忠实度、相关性、召回类指标和错误分析。

## 面试讲解点

- 多格式文档解析和清洗。
- Chunking 策略和参数选择。
- Embedding 与向量检索。
- Rerank 对召回质量的影响。
- 回答中的来源引用。
- Celery 异步处理上传文档。
- MinIO 对象存储。
- 为什么在线 Demo 要关闭注册、限制上传和隐藏基础设施端口。
